Summary

  • Hasil cetak Ricoh sangat sensitif / densitynya tinggi, sehingga terlihat lebih pekat dan cetakan timbul.
  • Compression Artefact terlihat di hasil cetak.
  • Paket yang didapatkan tanpa printer manager tambahan.
  • File vector yang diraster - TIFF hingga 1000dpi tetap menimbulkan compression artefact.
  • Menjaga kualitas setiap cetakan Docu Color.

  • (Contoh Compression Artefact)

End In Mind

  • Mencetak mengunakan file uncompressed / original file
  • Resolusi raster 600 ppi.
  • Setting bleed dari file original.
  • Memastikan efek gradasi diflat.
  • Mempertahankan image compression di 600 ppi pada hasil akhir PDF.

Source File dan Penanganannya

  • Semua proses layout menggunakan Adobe Indesign.

    Setting bleed dan crop mark dapat dilakukan di Illustrator / CorelDraw / Photoshop / Indesign

  • Hasil akhir berupa PDF.

  • Tipe file awal dan penanganan untuk Layout.

    Penanganan file awal untuk layout
    • Raw Illustrator » Save as Illustrator
    • Raw CorelDraw » Save as PDF prepress uncompressed
    • Raster file / Photoshop » Save as Photoshop 600 ppi
    • PDF » set bleed langsung di di Indesign atau cek sumber PDF dengan
      • Buka di Photoshop » Jika PDF Photoshop
      • Buka di Illustrator » Jika Vector

1. Workflow - Raw Illustrator File

Setting export PDF Prepress di Indesign

  • Preset: Press Quality.
  • Color output: No color conversion.
  • Compression: Bicubic Downsampling to 600 ppi.

Transparency flatten via Adobe Acrobat

  • Preset: High resolution.
  • 600 ppi gradient and mesh resolution.

2. Workflow - Raw CorelDraw File

Setting PDF Prepress CorelDraw

  • Compression: None.
  • Downsampling: 600 dpi.
  • Export all text as curves.

Setting export PDF Prepress di Indesign

lihat di setting PDF prespress indesign workflow illustrator.

Transparency flatten via Adobe Acrobat

lihat di transparency flatten Adobe Acrobat workflow illustrator.


3. Workflow - Raster File

Catatan
  • Pastikan image size 600 ppi.
  • Flatten image.
  • Save as Photoshop.
  • Setting Export PDF prepress sesuai workflow illustrator.

4. Workflow - PDF

4a. Semua setting dilakukan di Indesign

  • Proses bleed dan crop marks dilakukan di Indesign.
  • Jika saat proses print sangat berat, biasanya PDF photoshop dengan banyak layer (lakukan proses 4b).

4b. Cek PDF di Photoshop

  • Jika sumber file adalah PDF photoshop, maka akan terbuka layer-layer di photoshop.
  • Lakukan langkah seperti Workflow Raster File.
  • Jika saat dibuka hanya 1 layer, coba lakukan proses 4c.

4c. Cek PDF di Illustrator

  • Jika sumber file dari Illustrator / Corel, maka komponen design dapat terpisah-pisah.
  • Buat bleed dan crop marks di Illustrator, dan save as Illustrator.
  • Perhatikan gradasi / shadow, apabila terjadi masalah (warna kacau / gradasi menjadi raster) maka proses PDF di Photoshop (4b).
  • Penggunaan Illustrator untuk menjaga kestabilan warna cetak.
Knowledge
  • Dalam ujicoba, perbandingan antara kompresi Zip dan JPEG, hasil cetak terlihat lebih terang kompresi JPEG (Uji coba pada mesin Ricoh).
  • Bicubic Downsampling menggunakan rata-rata untuk menentukan warna piksel. Biasanya menghasillkan warna lebih tepat dan gradasi lebih smooth.
  • Proses Downsampling mengacu pada pengurangan jumlah piksel dalam suatu gambar.

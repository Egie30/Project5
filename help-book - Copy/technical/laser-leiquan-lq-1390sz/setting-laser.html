Setting - Laser

#laser #cutting #engraving

Persiapan

Sebelum melakukan proses cutting / engraving, pastikan setting power dan speed pada software / mesin sudah sesuai dengan tabel di bawah.

Perpaduan antara kecepatan dan tenaga (power) memberikan hasil cutting / engrave yang tepat. Kesalahan setting dapat mengakibatkan kurang putusnya cutting atau terbakarnya hasil cutting. Kesalahan ini juga mengakibatkan kurang dalamnya hasil engrave atau terlalu dalam.

Pastikan posisi head sudah fokus terhadap bahan. Lihat Cara memfokuskan head

Secara umum, setting untuk cutting : speed 25, power 35. Setting untuk engrave : speed 250 - 300, power 17 - 25, interval 0.06. Area kerja : 130x90 cm.

Catatan Penting

  • Engrave dengan ukuran kecil, gunakan power lebih besar dibandingkan ukuran besar. Apabila antara ukuran kecil dan besar disamakan powernya, maka hasil engrave ukuran kecil terlihat kurang dalam dibandingkan hasil engrave ukuran besar.
  • Engrave kaca. Sebelum proses, sebaiknya permukaan kaca yang di engrave diberi kertas yang dibasahi. Tujuannya adalah mengurangi pecahnya kaca yang diakibatkan panas karena laser.
  • Untuk memotong acrylic secara manual, speed pada mesin rubah sesuai ketebalan dan jenis bahan. Gunakan tombol pulse (untuk mengeluarkan laser) dan tombol arah (untuk menggerakkan head). Exp.: acrylic 1 cm - speed 5, HPL - speed 20.
  • Untuk proses transfer dari software ke mesin, secara otomatis akan dibuat mirror.
  • Urutan layer akan menentukan urutan proses.
  • Matikan angin pada saat gravir, nyalakan angin saat cutting.
  • Media potong harus kondisi rata, supaya mendapatkan titik fokus yang tepat â†’ hasil terpotong.
  • Memotong manual dengan kelurusan
    Gunakan arah kiri/kanan berbarengan dengan pulse.
    Sebelumnya set speed sesuai ketebalan bahan.
  • Cutting / engrave : file akan automatic mirror di mesin.
  • Urutan warna layer menentukan urutan proses, untuk merubahnya gunakan tab priority
  • Untuk mengecek waktu dan urutan cutting, gunakan preview (gambar layar)

Fungsi angin yg dihembus saat cut, berfungsi utk mencegah terjadinya api. Tetapi mengakibatkan hasil cutting bergerigi. Ini bisa diakali dg mematikan angin, saat keluar api di pause dulu terus jalan lagi, atau dikeluarkan anginnya. Atau kurangi kecepatan tanpa diangin. Memastikan api yg keluar tdk merusak.

  • Mesin tidak bisa terima data saat sedang proses kerja.
  • Suhu chiller : 20-35. Lebih dari 35 berbahaya.
